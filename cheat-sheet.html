Ruby Strings, Integers, Arrays, and Hashes

Strings
A string is a common data type in Ruby used to represent a collection of characters. A string can be defined using single or double quotes as follows:
"This is a string"
'This is also a string'
"Thisisstillastring110*3$%!??"

Integers
An integer is a common data type in Ruby used to represent a number. Integers may be positive or negative, but do not include decimal places:
1
876
-97
123456789876543210

Arrays
An Array is a common data structure in Ruby used to represent an ordered list of elements. Each element in the list can be referenced by its corresponding index number, starting with index 0. For example:
example_array = [first_element, second_element, third_element]
example_array[0] => returns first_element
example_array[2] => returns third_element

Hashes
A Hash is a common data structure in Ruby used to represent an unordered list of data. Each piece of data is stored as a pair of information, with a "value" that is referenced by it's corresponding "key". For example:
example_hash = {
a_key: a_value,
c_key: c_value,
b_key: b_value
}
